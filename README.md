# lain-get
Download files from imageboards.

## Requirements
* boost-libs
* libcurl
* jsoncpp

## Installation
```
git clone https://github.com/himnish/lain-get.git
cd lain-get
make
```

## Supported Forums
* [lainchan](https://lainchan.org)
* [4chan](https://4chan.org)
* [wizchan](https://wizchan.org)
